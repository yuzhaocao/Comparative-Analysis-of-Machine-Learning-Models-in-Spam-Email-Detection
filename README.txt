The "data" file contains 6 dataset which each consist of the "ham" and "spam" folder.
The "ham" folder contains legitimate emails and the "spam" folder contains the email spams

The "final_project.ipynb" contains jupyter notebook file, which includes the step by step how the program is being processed.
And also contains all the results run by the code.

The "project.py" contains the python file of the project as a whole.

Before running the project, make sure all the modules have been installed, these are all been stated on the top of "final_project.ipynb" and "project.py".
